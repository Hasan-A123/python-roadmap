# 📒 Project 6 - Contact Book (Python Roadmap)

A command-line Contact Book application built using Python dictionaries.

This project allows users to manage contacts using a simple menu-driven interface.

---

## 🚀 Features

- Add a contact
- View all contacts
- Search for a contact
- Delete a contact (with confirmation)
- Update a contact
- Exit the program

---

## 🧠 Concepts Practised

- Python dictionaries
- Loops (`while`)
- Conditional statements (`if/elif/else`)
- Input handling and validation
- CRUD operations:
  - Create
  - Read
  - Update
  - Delete
- Data organisation using key-value pairs
