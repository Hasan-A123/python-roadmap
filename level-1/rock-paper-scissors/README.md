## 🎮 How the Game Works

The player chooses one of three options:

- Rock
- Paper
- Scissors

The computer randomly selects one of the same three choices.

The winner is determined using the following rules:

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
- Matching choices result in a draw

The scoreboard is updated after each round.
