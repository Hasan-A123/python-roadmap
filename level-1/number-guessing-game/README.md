# Number Guessing Game

## Description
A simple command-line game where the player guesses a randomly generated number. 

## Features
- Random number generation
- Guess checking
- Too high / too low hints
- Score counter
- Play again option

## Concepts Practiced
- Variables 
- While loops
- If/elif/else
- User input
- random module

## Future Improvements
- Difficulty levels
- High score saved to a file
- Input validation
