# 🔐 Project 7 - Password Generator (Python Roadmap)

A command-line Password Generator built with Python.

Generate secure random passwords with customizable character options, including uppercase letters, numbers, and punctuation symbols.

---

## 🚀 Features

- Generate passwords of any length
- Include lowercase letters
- Optionally include uppercase letters
- Optionally include numbers
- Optionally include punctuation symbols
- Input validation for password length
- Input validation for yes/no options

---

## 🧠 Concepts Practised

- Modules (`random`, `string`)
- Strings
- Loops (`for`, `while`)
- Conditional statements
- Input validation
- String concatenation
- Random selection
- Building character pools

---

## ▶️ How to Run

```bash
python main.py
```
