# To-Do List

## Description

A command-line To-Do List application built in Python.

This project allows the user to manage a list of tasks through a simple menu.

- Lists
- Loops

- User Input
- Conditionals

## Features

- Add tasks
- View tasks
- Remove tasks
- Exit the application
